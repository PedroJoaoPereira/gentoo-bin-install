# Gentoo installation guide

Gentoo install with distribution kernel, LUKS + BTRFS, Dracut, Grub, doas, snapper and borg

## Prerequisites

- UEFI
- Internet connection
- Admin Live CD from Gentoo
- Partitioning tool
- Patience

---

Follow the guide from the `install` directory.
